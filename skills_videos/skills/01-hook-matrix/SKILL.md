# Hook Matrix 4x3 — Video Ad Creative System

## What This Skill Does
Generates a structured creative testing matrix combining 4 psychological hook angles with 3 visual formats, producing systematic ad variants instead of random ideas.

## The 4 Hook Angles

### Hook A — Pain Point
Target the specific frustration the audience feels RIGHT NOW.
- Open with the exact words the customer says when they complain
- Make it hyper-specific: not "your business isn't growing" but "you're paying for ads and getting zero messages"
- Formula: `"[Specific situation]? [Specific consequence they hate]"`
- Example: "¿Publicas todos los días y nadie te escribe? Eso es presupuesto que se va a la basura."

### Hook B — Transformation
Open with the RESULT, not the process.
- Show or describe the after-state in the first 2 seconds
- Use numbers, timelines, or before/after contrast
- Formula: `"De [before state] a [after state] en [timeframe]"`
- Example: "De 0 a 40 clientes por mes — sin agencia, sin gastar más."

### Hook C — Social Proof
Lead with external validation — numbers, names, or crowd behavior.
- "Más de X personas ya…"
- "[Type of person] está usando esto para…"
- "[Specific result] en [specific timeframe]"
- Formula: `"[Number/group] [achieved result] — [how]"`

### Hook D — Curiosity Gap
Create an information gap the brain is compelled to close.
- Make a statement that raises an obvious question
- Reference a counterintuitive insight or surprising data
- Formula: `"La razón por la que [common thing] no funciona es [unexpected answer]"`
- Example: "El error que comete el 90% de los negocios en sus primeras campañas de Meta."

## The 3 Visual Formats

### Format 1 — Product Hero
The product or result IS the protagonist from frame 1.
- Visual: product in use, close-up of result, demo in real context
- Voiceover or on-screen text carries the hook copy
- Best for: physical products, food, fashion, gadgets, before/after results
- Editing style: clean cuts, real environment, good lighting

### Format 2 — UGC / Testimonial
A real person (or creator) speaking directly to camera.
- Visual: selfie-style or slightly elevated, natural background
- Hook must come from the person's mouth in first 2 seconds
- Best for: services, apps, coaching, anything where trust = conversion
- Editing style: raw, real, minimal graphics, large captions

### Format 3 — Bold Typography
Text-first, visual second. Headline dominates the frame.
- Visual: strong background (product shot, b-roll, or color block) + large text overlay
- Hook is the headline — must work as standalone text
- Best for: SaaS, B2B, offers with strong numbers, faceless content
- Editing style: kinetic text, punchy cuts, minimal narration

## The 4x3 Matrix Output Structure

For each business, generate the matrix in this format:

| | Format 1: Product Hero | Format 2: UGC/Testimonial | Format 3: Bold Typography |
|---|---|---|---|
| **Hook A: Pain Point** | Hook A + Product Hero script | Hook A + UGC script | Hook A + Typography treatment |
| **Hook B: Transformation** | Hook B + Product Hero script | Hook B + UGC script | Hook B + Typography treatment |
| **Hook C: Social Proof** | Hook C + Product Hero script | Hook C + UGC script | Hook C + Typography treatment |
| **Hook D: Curiosity Gap** | Hook D + Product Hero script | Hook D + UGC script | Hook D + Typography treatment |

## Per-Cell Output (What to Generate for Each Variant)
```
Hook: [exact opening line — 1 sentence max]
Format: [Product Hero / UGC / Bold Typography]
0-2s: [what happens visually + what is said/shown]
2-5s: [development]
5-10s: [proof or mechanism]
10-15s: [offer + CTA]
On-screen text: [key text overlays]
CTA button: [exact text]
```

## Priority Combinations (Best Bets)
For PRODUCTS: Hook A + Format 1, Hook B + Format 2, Hook D + Format 3
For SERVICES: Hook A + Format 2, Hook C + Format 2, Hook D + Format 3
For SOFTWARE/SaaS: Hook D + Format 3, Hook B + Format 3, Hook A + Format 2

## A/B Test Priority
Always launch Hook A + Hook B first (Pain vs Transformation). Add Hook D if budget allows. Format 2 (UGC) consistently outperforms on TikTok; Format 3 (Typography) outperforms on Meta Feed desktop.
