# UGC Testimonial Engine

## Framework Logic
UGC ads win on trust metrics. A real person's authentic experience converts at 2-4x the rate of branded creative because it bypasses the viewer's "this is an ad" filter. This skill generates structured UGC scripts that feel authentic while ensuring every second serves a conversion purpose.

**Critical rule:** The product must never feel like the hero. The PERSON is the hero. The product is the tool that enabled their result.

---

## UGC Script Architecture

### Opening Line Types (first 2 seconds — choose one)

**Type 1 — Confession**
"La verdad es que al principio no le creí."
"Nunca pensé que esto me iba a funcionar a mí."
"Fui escéptica — no voy a mentirles."

**Type 2 — Result-first**
"Llevo 3 semanas usando esto y ya tengo [X resultado]."
"Esto me cambió el negocio en 30 días."

**Type 3 — Problem identifier**
"Si llevas meses pagando ads sin resultados, esto es para ti."
"¿Alguien más sentía que tiraba el dinero en publicidad?"

**Type 4 — Social observation**
"Todo el mundo en mi industria está usando [product] — y entiendo por qué."

## 3 UGC Script Variants (generate all 3)

### Variant A — The Skeptic → Believer
Best for: New products, B2B, high-ticket services
```
0-2s:  "No lo creía hasta que lo probé."
2-6s:  [Context: who they are + what they had tried before]
6-10s: [The moment of change + specific result]
10-15s: Natural recommendation + CTA
```

### Variant B — The Problem → Relief
Best for: Products/services with clear pain point
```
0-2s:  "[Specific problem stated as personal experience]"
2-6s:  [How long/how bad the problem was]
6-10s: [How the product solved it + tangible result]
10-15s: "Si tienes ese problema, prueba [product]."
```

### Variant C — The Result Showcase
Best for: Products with visible/measurable results
```
0-2s:  "[Lead with the result — number, transformation, achievement]"
2-6s:  [Context: before state briefly]
6-10s: [What they did / how they used it]
10-15s: "[Product name] + CTA"
```

## Visual Production Guide for UGC

### Camera Setup
- **Device:** Phone (not DSLR) — imperfection = authenticity
- **Position:** Eye level or slightly above, handheld or phone propped naturally
- **Framing:** Head and shoulders, speaker centered, environment visible
- **Background:** Real space — home, office, store, outdoor. NOT a plain wall.
- **Lighting:** Natural light from window. Ring light is acceptable but lower brightness.

### B-Roll Moments (insert between talking head shots)
- Product being picked up, opened, used
- Screen recording of app/software in action
- Before state (mess, problem, old method)
- Result/after state (clean, organized, numbers on screen)

### Captions (mandatory)
- Size: large (30% of frame width minimum)
- Style: bold, white with dark shadow or colored highlight on key words
- Every word of the script should be captioned
- Highlight emotion words with color

## Writing Authentic-Sounding Copy

### Phrases that sound real
- "la verdad es que…"
- "nunca pensé que…"
- "fue cuando me di cuenta de que…"
- "no sé si le pasa a alguien más pero…"
- "literalmente en [X días]…"
- "no lo esperaba pero…"

### Phrases that sound scripted (AVOID)
- "Este increíble producto…"
- "Estoy muy contento/a de compartir…"
- "Los resultados han sido fenomenales…"
- "Sin duda alguna te lo recomiendo…"
- Any superlatives: amazing, incredible, life-changing, best ever

---

## Batch Production Note
UGC scripts are designed for scale. One story can generate 3-5 script variants:
- Same story, different opening line
- Same content, different duration (15s / 30s / 60s)
- Same script, different person delivering it
- Same framework, different result metric emphasized

## Script Output Template

```
[TITLE]: UGC [Variant A/B/C] – [Business Name] – [Platform] – [Duration]s

LINE 1 (0-2s): "[exact opening — feel real, not scripted]"
DELIVERY NOTE: [tone — skeptical / relieved / excited / casual]

CONTEXT (2-6s): "[who they are + situation]"
B-ROLL: [optional cutaway]
ON-SCREEN: [key text]

RESULT (6-10s): "[specific result or transformation]"
VISUAL: [product in use or result shown]
ON-SCREEN: [metric or key phrase]

RECOMMENDATION + CTA (10-15s): "[natural rec + action]"
ON-SCREEN: [product name + CTA]
BUTTON: "[CTA text]"
```
