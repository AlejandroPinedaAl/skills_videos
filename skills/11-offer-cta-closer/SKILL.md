# Offer-CTA Closer

## Core Principle
Most AI-generated scripts stop at "awareness content." They describe the product beautifully and then say "Visítanos" or nothing at all. This skill exists to ensure every video ends with a specific, actionable, friction-reducing close that matches the campaign objective.

**Rule:** The CTA must tell the viewer exactly what to do, why to do it now, and what happens when they do.

---

## CTA Anatomy

```
[ACTION VERB] + [SPECIFIC DESTINATION] + [FRICTION REDUCER] + [URGENCY]

Example: "Activa" + "tu primera campaña" + "gratis" + "hoy"
= "Activa tu primera campaña gratis hoy."
```

### Action Verbs (by objective)
| Objective | Verb Options |
|---|---|
| Lead gen | "Escríbenos", "Solicita", "Pide", "Reserva", "Agenda" |
| Purchase | "Compra", "Pídelo", "Ordena", "Llévalo" |
| Trial/Signup | "Prueba", "Activa", "Regístrate", "Crea tu cuenta" |
| Content/Download | "Descarga", "Obtén", "Accede" |
| Awareness | "Síguenos", "Descubre", "Conoce" |

### Friction Reducers (always include one)
- "gratis" / "sin costo"
- "sin contrato"
- "en 2 minutos"
- "sin tarjeta de crédito"
- "prueba 14 días"
- "primer mes sin costo"
- "sin compromiso"

### Urgency Triggers (use when truthful)
- "hoy" / "esta semana"
- "plazas limitadas"
- "solo hasta el [date]"
- "los primeros 50"
- "antes de que suba el precio"

## CTAs by Business Type

### Product — E-commerce
```
Compra hoy con envío gratis.
Pídelo ahora — quedan [X] en stock.
Pruébalo 30 días — o te devolvemos el dinero.
Primer pedido con 20% de descuento.
```

### Service — Agency / Consulting
```
Agenda tu diagnóstico gratis esta semana.
Pide tu auditoría sin costo — link en bio.
Escríbenos "AUDIT" y te contactamos hoy.
Reserva tu sesión gratuita — solo 5 disponibles.
```

### SaaS / App
```
Empieza gratis — sin tarjeta de crédito.
Activa tu cuenta en 2 minutos.
Prueba 14 días gratis. Cancela cuando quieras.
Únete a [X] negocios que ya usan [product].
```

### Local Business
```
Reserva ahora — primeras 10 reservas con descuento.
Llama hoy y menciona este video.
Ven esta semana y presenta este anuncio.
Agenda tu cita gratis en [platform].
```

## CTA by Platform

| Platform | CTA Format | Constraints |
|---|---|---|
| TikTok | Verbal + text overlay + comment CTA | "Comenta [word]", "Link en bio" — no direct links in video |
| Meta Feed | Button CTA + verbal | Match button text to verbal CTA exactly |
| Meta Stories | Swipe-up or tap | "Desliza hacia arriba" or "Toca el enlace" |
| Google Bumper (6s) | On-screen text only — no time for verbal | Must appear in first 4s |
| Google Skippable | Verbal before 5s + button throughout | First 5s CTA must work if viewer skips after |
| YouTube | End card + verbal | Can reference timestamps or subscription |

## Funnel-Stage CTAs

### Top of Funnel (Awareness)
- Don't ask for purchase — ask for low-commitment action
- "Síguenos para más", "Guarda este video", "Comenta si esto te pasó"

### Middle of Funnel (Consideration)
- Move to information exchange
- "Pide info gratis", "Descarga la guía", "Ve el demo completo"

### Bottom of Funnel (Conversion)
- Direct purchase or high-intent action
- "Compra ahora", "Activa hoy", "Agenda tu sesión"

## The Offer Frame (2-3s before the CTA)
Before the CTA, briefly state the offer. This is separate from the CTA itself.

```
OFFER FRAME: "[What they get] + [at what cost/risk]"
CTA: "[What to do to get it]"

Example:
OFFER: "Tu primera campaña completamente gratis — sin contrato."
CTA: "Actívala ahora en SaleADS."
```

## Script Output Template

```
[TITLE]: CTA Block – [Business Name] – [Platform] – [Duration]s

OFFER FRAME (X-Xs):
COPY: "[what they get + friction reducer]"
VISUAL: [product/platform + offer text on screen]
ON-SCREEN: "[offer text bold]"

CTA (X-Xs):
VERBAL: "[exact words]"
ON-SCREEN: "[same words as large text]"
BUTTON: "[platform-native CTA button text]"
URGENCY: [yes/no — if yes, what]
```
