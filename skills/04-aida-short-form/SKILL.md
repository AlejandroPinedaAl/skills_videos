# AIDA Short-Form — Attention → Interest → Desire → Action

## Framework Logic
AIDA is the most universal sales framework because it mirrors how decisions are made: something grabs us, we want to know more, we imagine having it, and then we act. In short-form video, each phase gets compressed — but the sequence is non-negotiable.

**Key difference from PAS:** AIDA doesn't start with pain. It starts with a bold statement or claim that earns attention independently. Use AIDA when the product/service is strong enough to sell on desire, not just pain relief.

---

## Structure (15-second version)

```
ATTENTION   0 – 3s
├── Bold claim, surprising stat, or strong question
├── Must work with sound OFF (on-screen text carries it)
├── No "Hi, I'm…" — start with the strongest sentence you have
└── Examples:
    "La mayoría de los negocios malgasta el 60% de su presupuesto en ads."
    "¿Y si pudieras saber exactamente qué anuncio va a funcionar antes de publicarlo?"
    "Esto cambió cómo generamos clientes para siempre."

INTEREST    3 – 7s
├── Explain the HOW or WHAT — the mechanism that makes it real
├── One differentiating detail that separates from competition
├── Visual: product feature, dashboard, process, comparison
└── Example: "SaleADS analiza tu negocio, tu audiencia y el mercado —
             y genera el creativo por ti, adaptado a cada plataforma."

DESIRE      7 – 12s
├── Paint the outcome — the life/business AFTER using this
├── Make it sensory or concrete: numbers, time saved, feeling
├── Social proof if available: "más de X negocios ya…"
└── Example: "Imagina lanzar una campaña en 10 minutos que te trae leads
             calificados mientras duermes."

ACTION      12 – 15s
├── Single CTA — no choices
├── Remove friction: "gratis", "sin contrato", "en 2 minutos"
└── Example: "Empieza gratis hoy — sin tarjeta."
```

---

## Structure (30-second version)

```
ATTENTION   0 – 5s    Stronger opening — can include a micro-story setup
INTEREST    5 – 12s   Two differentiating details, mini-demo
DESIRE      12 – 22s  Result + testimonial clip or data point
ACTION      22 – 30s  CTA + offer + urgency
```

## Attention Formulas by Type

| Type | Formula |
|---|---|
| Stat-based | "[X]% de [audience] [does bad thing]. ¿Eres uno de ellos?" |
| Question | "¿Qué pasaría si [desired outcome] fuera tan fácil como [simple action]?" |
| Bold claim | "[Product/service] es la razón por la que [type of person] [achieves result]." |
| Counterintuitive | "No necesitas [common belief]. Lo que realmente necesitas es [truth]." |
| Achievement | "Cómo [specific person/type] logró [specific result] en [timeframe]." |

---

## AIDA by Business Type

### Product Launch
- **A:** Biggest benefit stated as fact
- **I:** How it works (1 key feature)
- **D:** Transformation or lifestyle outcome
- **A:** "Pídelo ahora" / "Envío gratis hoy"

### Service / Agency
- **A:** Surprising insight about the client's current situation
- **I:** The methodology or approach
- **D:** Client result (specific, named if possible)
- **A:** "Agenda tu diagnóstico gratis"

### App / SaaS
- **A:** Time or money stat
- **I:** Core feature in 1 sentence
- **D:** Dashboard/result screenshot + user count
- **A:** "Pruébala gratis por 14 días"

## Platform Tuning

| Platform | Key Adjustment |
|---|---|
| Meta Feed | Attention block must work silent (text-first). AIDA can stretch to 30s. |
| Meta Stories | Compress to 10-15s. Large text. Vertical. CTA with swipe-up language. |
| YouTube Pre-roll | First 5s = ATTENTION. Everything before skip button must earn the watch. |
| TikTok | Add pattern interrupt before A. Delivery faster. Captions mandatory. |
